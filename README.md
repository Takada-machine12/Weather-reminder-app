# Stock-reminder-app
twelve dataのAPIを使用してお気に入り登録した株の株価情報を自分のメールに通知するWebアプリ

# 機能一覧
### 【ユーザー側の機能】
- ユーザー登録
- ログイン機能
- パスワード再発行機能
- ユーザ情報編集機能
- 退会機能
- お気に入り株登録機能
- お気に入り株一覧機能
- 株価通知設定機能
- メール通知設定機能
- 管理者からのお知らせ情報を表示する機能
- 通知履歴機能
### 【バッチ処理】
- 通知実行機能
### 【管理者側の機能】
- 管理者画面のログイン機能
- ユーザ一覧表示機能
- ユーザ情報編集機能
- ユーザへのお知らせ通知設定
