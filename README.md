# Weather-reminder-app
天気のAPIを使用して登録した地域のお天気情報をメールに通知するWebアプリ

# 機能一覧
### 【ユーザー側の機能】
- ユーザー登録
- ログイン機能
- パスワード再発行機能
- ユーザ情報編集機能
- 退会機能
- お気に入り天気登録機能(自分の好きな地域を登録　例：東京都千代田区〇〇町)
- お気に入り天気一覧機能(登録した地域とその地域の天気の一覧を表示　削除できる機能も追加)
- 天気情報通知設定機能（例：〇〇℃以下になったら通知させるetc…）
- メール通知設定機能（設定できる時間幅は1時間刻みで設定できる）
- 管理者からのお知らせ情報を表示する機能
- 通知履歴機能（設定した通知時間に通知が完了すると表示される）
### 【バッチ処理】
- 通知実行機能（設定した通知を実行するファイル　API取得、条件判定、メール送信）
### 【管理者側の機能】
- 管理者画面のログイン機能
- ユーザ一覧表示機能
- ユーザ情報編集機能
- ユーザへのお知らせ通知設定
